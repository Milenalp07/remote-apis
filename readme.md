# Remote APIs Project

This project is a simple web application built using **HTML, CSS, and JavaScript**.
It demonstrates how to fetch and display data from remote APIs using the **Fetch API with async/await**.

## Project Overview

The page contains **three buttons**, each of which fetches data from a different public API and displays the results in a **grid layout**.

The application also includes:

* A **region dropdown** to search for countries by region.
* A **search box** to search for Rick & Morty characters.
* **Error handling** for API requests.

## APIs Used

The following public APIs are used in this project:

### Countries API

https://restcountries.com/v3.1/region/europe

This API returns information about countries such as:

* Name
* Capital
* Population
* Flag

### Users API

https://jsonplaceholder.typicode.com/users

This API provides fake user data including:

* Name
* Email
* Phone number
* Company

### Rick and Morty API

https://rickandmortyapi.com/api/character

This API provides character information including:

* Name
* Status
* Species
* Character image

## Features

* Fetch data from remote APIs
* Use **async/await syntax**
* Display results dynamically in a **grid layout**
* Dropdown menu for selecting a **country region**
* Search box for **Rick & Morty characters**
* Basic **error handling**
